# Scaler AI Projects

AI applications built during the Scaler AI Engineering Program using free APIs, Python, and Gradio.

## Projects

### 1. 🔎 AI Website Summarizer
Enter any website URL and get an AI-generated markdown summary.
- **Stack:** Python, BeautifulSoup, Groq API (Llama 3.3 70B), Gradio
- **Folder:** `ai-website-summarizer/`

### 2. ⚔️ LLM Arena
Send the same prompt to two LLMs and vote on which answered better.
- **Stack:** Python, Groq API (GPT-OSS 120B vs Llama 3.1 8B), Gradio
- **Folder:** `llm-arena/`
